# How to Use the Code

Follow these steps to get started with the project:

1. **Download the repository**  
   Clone or download this repo to your local machine.

2. **Run the code in your IDE**  
   You can use IntelliJ or any IDE you prefer. The source code is located here:  
   [src folder](https://github.com/Abanoub-Adel-Gerges/Text-Search/tree/main/Text%20Search/src)

3. **Follow the instructions**  
   Open the `How to use.pptx` file and follow the steps carefully.

4. **Choose your interface**  
   You can either use the **console project** or the **GUI version**, whichever you prefer.
